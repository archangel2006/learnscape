# **App Name**: Learnscape

## Core Features:

- Responsive Landing Page UI: Build the complete, responsive UI for the marketing-oriented landing page, incorporating the Hero, How-It-Works, Features, Demo Preview, Technology Stack, and Call-to-Action sections.
- Demo Page UI with Camera Feed: Implement the core UI for the Demo page (/scan), including the top bar, main camera preview container, a transparent overlay layer, and the bottom control panel, displaying the device's live camera feed.
- Interactive Overlay Placeholder: Establish a transparent overlay system above the camera feed on the Demo page, designed to visually accommodate future AI-generated diagrams, annotations, and visual explanations directly on scanned objects.
- Generative AI Voice Interaction Placeholder: Integrate a microphone UI element in the Demo page's control panel, acting as a placeholder to allow users to interact with a future generative AI model to ask questions about the scanned objects and receive explanations.
- Theming System with Toggle: Develop a dual-theme system (Light Mode and Dark Mode) with a user-facing toggle for switching between themes, applying soft off-white and deep dark backgrounds respectively, and adjusting UI element colors.
- Seamless Page Navigation: Enable smooth client-side navigation between the Landing Page (/) and the Demo Page (/scan) via specified button actions.

## Style Guidelines:

- Primary accent color: A modern, vibrant blue (`#2663D9`), embodying intelligence and technology. This color will be used for interactive elements, highlights, and calls to action.
- Light theme background: A soft, almost off-white hue (`#F9F9FB`), subtly tinted with the primary blue, ensuring a clean and inviting appearance for educational content.
- Dark theme background: A deep, near-black shade (`#17181C`), subtly tinted with the primary blue, providing a focused and sleek environment reminiscent of modern AI product interfaces.
- Secondary accent color: A lighter, analogous cyan-blue (`#84D1E0`) provides visual distinction for supporting elements while maintaining a coherent, futuristic palette.
- Headline and body font: 'Inter' (sans-serif), chosen for its modern, clean aesthetic, excellent legibility, and versatility across both informative headlines and detailed explanations, suitable for a wide audience.
- Utilize a set of minimalist, clean line icons that communicate concepts effectively without distracting from the primary content. Icons for camera, microphone, and learning concepts should be clearly discernible.
- Implement a fully responsive grid-based layout with generous spacing, ensuring optimal readability and usability across all devices (mobile, tablet, desktop) while maintaining content hierarchy and a visually engaging aesthetic. Content should feel 'airy' and uncrowded.
- Incorporate subtle, tasteful transitions and micro-interactions (e.g., button hovers, page transitions, modal opening) to enhance user experience, giving the application a polished, 'modern AI product' feel without being distracting.
